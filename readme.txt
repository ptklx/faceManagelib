操作说明：


method ：  是选择标7个点（1）还是68个点（2）还是所有点（3）

start： 是初始化加载数据，首先需要加载

test： 是查看标记是否正确，

stepnum：是步长，
 

skip：是跳过这张图，不标记，

fix：是修正图


next：是查看自动标记是否正确，如不正确请fix


facePath：  是图片路径

outPath：是生成txt的标记保存路径

在执行路径下./modleDate/lastReadLoc.txt 
中是上次执行的第几个人剩下几张图片，0_2_46    标识第2个人，还剩46张没标记



其中，fix 时，是另建立窗口，退出可按q 或Esc，  按d，是删除全部点，重新标记

上下左右移动点按键是小键盘 2 ，4，6，8 按键

按 s，是保存当前标记的图，在执行目录下

当需要更改单个点时，按住ctrl键，左击相应的点，（图片可以放大）

 + - 键是放大和缩小

v 是上下放大，b是左右放大，  n 是上下缩小， m是左右缩小

注意，退出不能点右上角X，只能按q或Esc

其中test，查看图片，除了q 和Esc键是退出，其他都是浏览下一张，


由于图片比较多，每次新加载一个人时会比较慢，请等待……
